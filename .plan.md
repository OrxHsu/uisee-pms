# 缺失物料卡片样式调整

## Context
用户要求调整「车辆到场 → 缺失物料」页面中物料卡片的两处样式：
1. 卡片底色改为粉色（当前为橙色）
2. 编辑和删除按钮垂直居中（当前顶部对齐，视觉上偏高）

## Changes

### File: `src/components/modules/vehicle-arrival/VehicleArrivalModule.tsx`

#### 1. 卡片底色改为粉色
- **Location**: `MissingContent` 组件中卡片容器（line ~445）
- **Current**: `bg-orange-50 dark:bg-orange-900/20`
- **New**: `bg-pink-50 dark:bg-pink-900/20`
- **Rationale**: 使用 Tailwind 的 pink 色板，保持与暗色模式的对应关系

#### 2. 按钮垂直居中
- **Location**: `MissingContent` 组件中卡片内层 flex 容器（line ~446）
- **Current**: `flex items-start justify-between`
- **New**: `flex items-center justify-between`
- **Rationale**: `items-center` 使按钮与左侧内容垂直居中对齐，解决"太高"的视觉问题

## Verification
1. 启动 dev server (`npm run dev`)
2. 在预览浏览器中进入「车辆到场 → 缺失物料」
3. 确认卡片底色为粉色
4. 确认编辑和删除按钮与卡片内容垂直居中
